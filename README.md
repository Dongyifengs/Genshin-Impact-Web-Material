## ✨ Genshin Impact Web Material是什么项目？
- `Genshin Impact Web Material`是我一个人在维护的一个项目,该项目的目的是为了方便大家使用原神活动H5页面上有关Spine动效,并且提供了一些常用的工具,例如:Spine动效的预览,Spine动效的下载,Spine动效的导出等等.
