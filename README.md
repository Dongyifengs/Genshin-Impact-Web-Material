## ✨ mihoyo-material是什么项目？
- `mihoyo-material`是一个原神活动H5页面上有关Spine动效整理
- 里面包含`有关当前H5页面所有的素材文件`,`Spine`,`atlas`文件的提供
- 还有人物的拆分图层.

## 💡️ mihoyo-material该如何使用？
- 打开Spine软件
- 点击导入项目,选择对应的`json`文件
- 然后选择纹理解包器,选择atlas文件
